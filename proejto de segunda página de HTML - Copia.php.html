Sobre Html

Essa é uma pagina criada para esplicar Tags HTML e suas funções

  • < DOCTYPE > - A Primeira tag a ser usada para iniciar um documento html é a tag DOCTYPE, a tag doctype foi criada para definir para o navegador o tipo de documento que ele terá que interpretar será criado.
  • < Html > - A tag html define que tipo de código de arquivo será escrito.
  • < Head > - Define o campo de backzone de um documento.
  • < Body > - Define o campo de corpo dá página ou seja o que será exibido para o leitor a olho nu.

    • As tags acima junto com seus respectivos fechamentos são o principio de uma página html.

    Abaixo algumas tags vista em sala de aula.

  • < Title > - Define um título para a página.
  • < Meta > - Define uma meta para a página.
  • < Charset > - Define o formato de codificação de caracteres.ASCII, UTF-8, ANSI e ISO-8859-1 são exemplos de charsets.
  • < h1> - É uma tag que define um titulo váriavel de h1 até h6 sendo h6 o grau de menos relevancia.
  • < P > - Adiciona um parágrafo.
  • < br > - Adiciona uma quebra de linha na página ou seja um linha em branco.
  • < b > - Toda palavra ou frase digitada dentro dessa tag sera em negrito.
  • < em > - Tranforma um texto em itálico
  • < small > - Toda palavra ou frase escrita dentro dessa tag vai ser menor que as demais frases exemplo : frase test
  • < Strong > - Dá enfaze maior a escrita.
  • < Figure > - Adiciona parâmetros que permitem adicionar uma imagem.
  • < !----> - Adiciona um comentario ao código, esse comentário não pode ser visto no navegador, somente na linha de código.
  • < A > - Define um espaço para um hiperlink,ou seja uma ancora para que seja adicionado um caminho para outro site, exemplo : moranguinho
  • < Table > - Cria uma tabela, acompanhado de outras duas sub tags que são < tr > e < td > a sua junção cria uma tabela como essa :
    culuna 1 coluna 2 coluna 3
    linha 1 linha 1 linha 1
    linha 2 linha 2 linha 2
  • < lu > - Frases escritas dentro desta tag acompanha da sub tag < li > será organizada em uma lista não numerada chamada de "lista não ordenada", exemplo
  • linha 1
  • linha 2
  • linha 3
    •  o mesmo formato usado para o listamentos das tags acima
  • < ol > - Frases escritas dentro desta tag acompanhada da sub tag < li > será organizada em uma lista numerada chamada "Lista ordenada".
